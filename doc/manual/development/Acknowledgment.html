Software

Client

  • three.js
    • NGL relies on the three.js library for using WebGL
    • NGL's documentation uses the three.js documentation as a template
    • NGL's [page:Interface GUI] is based on the three.js editor UI
  • sprintf.js - used for formatting text
  • jsfeat - the SVD code for the superposition method is from jsfeat
  • QUnit (assert-close) - used for [link:../../../test/unit/unittests.html testing]
  • Benchmark.js for benchmarking
  • Chroma.js for color handling
  • FlexiColorPicker for color picking
  • Virtual DOM List
  • Font Awesome for icons
  • JS Signals
  • tether.js
  • Lightweight promise polyfill
  • bzip2.js
  • JSZip
  • js-lzma
  • pako - zlib port

Server

  • MDAnalysis - NGL relies on the indexing and seeking capabilities of the libxdrfile2 library for accessing .xtc trajectory files
  • Flask - used to create the webserver
  • Python - programming language used for the webserver
  • NumPy - fast handling of numerical data, here trajectory files

Shader code

  • Open Source PyMOL - screen aligned cylinder shader
  • VTK Quadric shader code from the PointSprite Plugin - quadric surface center calculation
  • HyperBalls - hyperball stick shader - Chavent, M., Vanel, A., Tek, A., Levy, B., Robert, S., Raffin, B., & Baaden, M. (2011). GPU-accelerated atom and dynamic bond visualization using hyperballs: a unified algorithm for balls, sticks, and hyperboloids.
